高中生與動物睡眠習慣分析

使用線性回歸模型

1. 研究背景

  • 高中生睡眠問題
    • 學業壓力下的睡眠不足問題。
    • 如何提高睡眠質量？
  • 睡眠結構的影響
    • 深睡、淺睡對精神狀態的重要性。
    • 探索改善方法。

2. 實驗目標

  • 主目標
    • 探討睡眠結構對高中生當日精神狀態的影響。
  • 副目標
    • 找出深睡比例、淺睡比例與精神狀態的關係。
    • 提供改善高中生睡眠質量的建議。

3. 實驗設計

3.1 數據收集

高中生數據

  • 來源：問卷調查或智能穿戴設備（如手環、睡眠追蹤器）。
  • 變數：
    • 睡眠總時長（Total Sleep Duration, 小時）
    • 深睡時長（Deep Sleep Duration, 小時）
    • 淺睡時長（Light Sleep Duration, 小時）
    • 當日精神狀態（Daytime Energy Level, 1-10 等級）
    • 控制變數：
      • 年齡
      • 課業壓力（1-10 等級）
      • 運動時間（分鐘）

3.2 數據處理

  1. 數據清洗
    • 剔除不完整或異常值（如深睡時間 > 總睡時間）。
  2. 變數標準化
    • 將所有變數進行 Z-score 標準化。
  3. 特徵工程
    • 計算「深睡比例」和「淺睡比例」：

深睡比例=深睡時長睡眠總時長深睡比例 = \frac{\text{深睡時長}}{\text{睡眠總時長}} 深睡比例=睡眠總時長深睡時長​

淺睡比例=淺睡時長睡眠總時長淺睡比例 = \frac{\text{淺睡時長}}{\text{睡眠總時長}} 淺睡比例=睡眠總時長淺睡時長​

3.3 模型選擇

  • 回歸模型：多元線性回歸
    使用以下回歸方程進行建模：

    Y=β0+β1X1+β2X2+β3X3+ϵY = \beta_0 + \beta_1X_1 + \beta_2X_2 + \beta_3X_3 + \epsilonY=β0​+β1​X1​+β2​X2​+β3​X3​+ϵ

  • ( Y )：當日精神狀態

  • ( X_1 )：睡眠總時長

  • ( X_2 )：深睡比例

  • ( X_3 )：淺睡比例

3.4 評估指標

  • 模型性能
    • ( R^2 )：判定係數
    • MAE（平均絕對誤差）
  • 顯著性檢驗
    • 檢查每個變數的回歸係數是否顯著（p 值 < 0.05）。

4. 結果與可視化

4.1 數據分析

  1. 相關性分析
    • 探索深睡比例、淺睡比例與當日精神狀態的相關性。
  2. 模型構建與性能評估
    • 使用 Python 進行數據訓練與測試（80% 訓練集、20% 測試集）。

4.2 結果可視化

  • 相關性散點圖
    • 總睡眠時間 vs 當日精神狀態
    • 深睡比例 vs 當日精神狀態
  • 睡眠結構條形圖
    • 深睡比例與淺睡比例的分佈對比

5. 結論與建議

5.1 結論

  • 深睡比例對當日精神狀態具有更顯著的影響。
  • 總睡眠時間不足可能削弱深睡對精神狀態的正面效應。

5.2 建議

  • 改善深睡比例的方法：
    1. 適當的睡前運動。
    2. 調整睡眠環境（如光線、噪音）。
  • 進一步研究方向：
    • 探討飲食、心理壓力等其他因素對深睡的影響。
